- GNU Debugger
- Argument `-g` beim Kompilieren

> Nur zum debuggen verwenden, da executable durch debugging info vergrößert wird


> Run
- `run`
- `r [args]`
- run again to restart debugging process

> Breakpoints
- `break [line]`
- `b [file]:[line]`
- `b [func]`
- `info break`
- `d(elete)`
- `d [b num]`
- `disable [b num]`
- `enable [b num]`

> Printing
- `print [var/ptr]`
	- p ptr returns address
- `p [reg]`

- `x [ptr]`
	- prints value of ptr
- `x ([ptr + i])`
	- i is array index
- `x/[n] (ptr + i)`
	- n is amount of elem to be shown
- `x/[n][datatype] ptr`
	- show n elem of array at ptr in given format
	- `a` address
	- `s` string
	- `d` int
	- -> x uses <u>last used format</u>! 

> Step
- `s` step into
- `si` next assembly instruction
	- s and si same function in assembly code
- `n(ext)` step over
- `c(ontinue)` run until next breakpoint
